pyEntropy
=========

pyEntropy is a Python module for estimating entropy and information theoretic
quantities using a range of bias correction methods.

You can download the latest version from the project `home page
<http://code.google.com/p/pyentropy>`_.

Please direct comments/questions/feedback to ``pyentropy AT robince.net``.
